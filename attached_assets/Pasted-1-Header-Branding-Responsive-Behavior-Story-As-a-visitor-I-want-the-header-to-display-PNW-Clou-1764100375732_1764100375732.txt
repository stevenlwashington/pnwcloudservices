1. Header Branding & Responsive Behavior
Story

As a visitor, I want the header to display “PNW Cloud Services” next to the icon and link back to the homepage, so I always know whose site I’m on and can easily navigate home.

Acceptance Criteria

Icon + text behave as one unified clickable home link.

Desktop full width shows “PNW Cloud Services.”

At screen width < 900px, text changes to “PNW Cloud.”

On mobile, always display “PNW Cloud Services.”

Typography, spacing, color consistent with brand.

No wrapping, clipping, or overflow at any resolution.

Assumptions

Header uses global styling across all pages.

Business Value

Clear branding, consistent navigation, improved professionalism and trust.
